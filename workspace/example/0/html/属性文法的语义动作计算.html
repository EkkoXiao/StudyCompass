属性文法的语义动作计算

Definition

通过给定的产生式及其对应的语义动作规则，展示如何对特定语言进行语义计算

Information

本部分通过例子展示了如何利用属性文法的产生式以及与之相关联的语义动作来识别特定的语言L = { aibjck | i, j, k ≥ 1}，并且针对 anbncn (n ≥ 1) 这种特定形式的字符串进行合法性判断。具体的语义动作包括了对属性num的计算以及在满足特定条件时输出“Accepted!”或“Refused!”的动作。

Usage

用于识别和校验给定语言的字符串，并进行相应的语义处理
